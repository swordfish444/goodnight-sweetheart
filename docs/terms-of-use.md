# Terms of Use for Goodnight Sweetheart

Effective date: March 8, 2026

Goodnight Sweetheart is provided as a bedtime reminder and affirming Alexa experience. By enabling or using the skill, the customer agrees to use it through supported Alexa devices and Alexa reminder surfaces.

## Intended use

Goodnight Sweetheart is designed to help customers schedule affectionate bedtime reminders. It is not medical advice, mental health advice, crisis support, or emergency assistance.

## Reminder delivery

The skill uses Alexa reminders. Reminder delivery depends on:

- Alexa services and reminder permissions
- device availability
- the customer's device settings, including time zone configuration
- AWS and the external text-generation provider being available when reminders are created or updated

The generated reminder wording is created when a reminder is scheduled or updated. Alexa later delivers that stored reminder through the Alexa reminder surface.

## Customer controls

Customers control the schedules they create through the skill. Customers can ask Alexa to change or clear reminders at any time.

## External processing

The skill may use an external text-generation provider through OpenRouter to create short reminder wording. By using the skill, the customer understands that limited request context needed for the bedtime message may be sent to that provider as described in the privacy policy.

## Availability

The skill is provided on an as-is and as-available basis.

## Limitation of use

The skill should not be used for emergency, safety-critical, or time-sensitive obligations where a missed reminder could cause harm.
