# Privacy Policy for Goodnight Sweetheart

Goodnight Sweetheart is an Alexa custom skill that helps customers schedule affectionate bedtime reminders. This policy explains what information the skill uses and how that information is handled.

## Information the skill uses

Goodnight Sweetheart uses only the information needed to create, read, update, and delete the Alexa reminders requested by the customer.

This can include:

- the bedtime schedule the customer asks Alexa to create
- Alexa-generated reminder identifiers needed to manage reminders created by the skill
- the Alexa device time zone needed to schedule the reminder at the requested local time

## Information the skill does not request

Goodnight Sweetheart does not use account linking and does not request the customer's name, email address, phone number, or physical address.

## How information is used

The skill uses reminder-related information only to:

- create bedtime reminders requested by the customer
- read existing reminders created by the skill
- update or delete reminders when the customer changes or clears a schedule

## Sharing and advertising

Goodnight Sweetheart does not sell customer information. The skill does not use customer information for advertising.

## Data retention

Goodnight Sweetheart does not maintain a separate customer marketing database. Reminder-related information is used only as needed to manage Alexa reminders created through the skill. If a customer clears reminders, the skill removes the reminder configuration it manages through the Alexa Reminders API.
