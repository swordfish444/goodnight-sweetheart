# Privacy Policy

Goodnight Sweetheart stores and uses only the minimum information required to create, update, read, and delete Alexa reminders that the customer asks for through the skill.

## Data used

- Reminder schedule details the customer requests through the skill
- Alexa-generated identifiers required to manage reminders created by this skill

## Data sharing

Goodnight Sweetheart does not sell customer data and does not use customer data for advertising.

## Data retention

The skill only keeps reminder-related information needed to manage reminders created through Alexa. If a customer clears reminders, the skill removes the reminder configuration it controls through the Alexa Reminders API.
